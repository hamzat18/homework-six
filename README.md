## Getting Started

1. **Initialize the project**

   ```bash
   npm init -y
   ```

2. **Install Sass**

   ```bash
   npm install sass --save-dev
   ```

3. **Add Scripts**
   - Live Server:

   ```bash
   "serve": "live-server"
   ```

   - Compile a single SCSS file to CSS:

   ```bash
   "compile:sass": "sass scss/styles.scss css/styles.css -w --no-source-map"
   ```
